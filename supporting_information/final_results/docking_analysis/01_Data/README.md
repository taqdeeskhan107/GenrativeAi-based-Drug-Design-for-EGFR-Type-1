# Selected Molecules Dataset

This directory contains data for 5 selected molecules from the BindingForge project.

## Files

1. `bindingforge_selected_molecules.csv`
   - Contains basic information about the 5 selected molecules
   - Fields: mol_id, smiles, source, project_phase, date

2. `selected_molecules_properties.csv`
   - Contains calculated molecular properties
   - Fields: mol_id, molecular_weight, logp, hbd, hba, tpsa, rotatable_bonds, aromatic_rings

## Selected Molecules

The dataset includes:
- 2 molecules from Previous_Project (Phase 1)
- 3 molecules from BindingForge_Novel (Phase 2)

### Molecule Details:
1. Molecule 2: Previous project Phase 1 compound with cyclobutyl substituent
2. Molecule 3: Previous project Phase 1 compound with morpholine ring
3. Molecule 16: Novel compound with F/Cl substitution
4. Molecule 17: Novel compound with basic scaffold
5. Molecule 18: Novel compound with methoxy substitution
